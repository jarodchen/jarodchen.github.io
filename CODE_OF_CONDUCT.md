# 行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺：无论年龄、体型、残疾、种族、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性认同和性取向如何，每个人在参与我们的社区时都不会受到骚扰。

## 我们的标准

### 积极行为示例

- 使用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

### 不可接受的行为

- 使用性化的语言或图像，以及不受欢迎的性关注或挑逗
- 恶意挑衅、侮辱性或贬低性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如物理地址或电子邮件地址
- 其他可合理视为不适当的专业行为

## 我们的责任

项目维护者负责澄清可接受行为的标准，并预计将对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权且有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并暂时或永久禁止他们认为不适当、威胁、冒犯或有害的任何行为的贡献者。

## 适用范围

本行为准则适用于所有项目空间，以及在个人代表项目或其社区的公共空间中展示个人时。代表项目的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发帖，或在线上或线下活动中担任指定代表。

## 执行

如有滥用、骚扰或其他不可接受的行为，可通过 [contact@jarodchen.dev](mailto:contact@jarodchen.dev) 联系项目团队进行报告和审查。所有投诉都将得到及时和公平的审查和调查。

项目团队有义务对任何事件的报告者保密。有关具体执行政策的更多详细信息将单独发布。

未真诚遵守或执行本行为准则的项目维护者可能会面临其项目负责人决定的临时或永久性影响。
