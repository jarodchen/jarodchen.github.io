# 贡献指南

欢迎为项目做出贡献！

## 报告问题

发现 bug 或有功能建议时：
1. 先查看现有 Issues 是否有重复
2. 创建新 Issue，包含清晰的问题描述和复现步骤

## 提交代码

1. Fork 仓库并克隆到本地
2. 创建分支：`git checkout -b feature/your-feature`
3. 进行修改并提交
4. 推送到远程并创建 Pull Request

## PR 要求

- 代码能正常运行
- 遵循现有代码风格
- PR 描述清晰说明更改内容
- 使用语义化提交信息（feat:、fix:、docs: 等）

## 开发流程

```bash
git clone https://github.com/jarodchen/jarodchen.github.io.git
cd jarodchen.github.io
npm install
npm run dev
```

## 联系方式

有问题请创建 [Issue](https://github.com/jarodchen/jarodchen.github.io/issues)
