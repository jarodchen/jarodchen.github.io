# Jarod Chen's GitHub Pages

我的个人网站，记录技术学习历程、项目实践和知识分享。

## 🚀 在线访问

**[https://jarodchen.github.io/](https://jarodchen.github.io/)**

## 📦 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
