# RSS 订阅

## 📡 关于 RSS 订阅

RSS（Really Simple Syndication）是一种内容订阅协议，允许您通过 RSS 阅读器实时接收网站更新。

## 订阅地址

```
https://jarodchen.github.io/feed.rss
```

## 如何使用

### 1. 选择 RSS 阅读器

推荐使用以下 RSS 阅读器：

- **Feedly** - 跨平台，界面友好
- **Inoreader** - 功能强大，支持规则过滤
- **The Old Reader** - 简洁易用
- **Reeder** (macOS/iOS) - 原生应用体验佳

### 2. 添加订阅

在您的 RSS 阅读器中：

1. 点击"添加订阅"或"Add Feed"
2. 输入订阅地址：`https://jarodchen.github.io/feed.rss`
3. 确认订阅

### 3. 开始阅读

订阅成功后，每当本站发布新文章，您的 RSS 阅读器会自动推送更新通知。

## 技术说明

本站使用 [vitepress-plugin-rss](https://github.com/danlsgiga/vitepress-plugin-rss) 自动生成 RSS feed，包含所有博客文章的标题、描述、发布日期和链接信息。
