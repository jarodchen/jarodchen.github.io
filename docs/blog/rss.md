# RSS 订阅

## 📡 关于 RSS 订阅

RSS（Really Simple Syndication）是一种内容订阅协议，允许您通过 RSS 阅读器实时接收网站更新。

## 🔗 订阅地址

```
https://jarodchen.github.io/feed.rss
```

## 📖 如何使用

### 1. 选择 RSS 阅读器

推荐使用以下 RSS 阅读器：

- **[Fluent Reader](https://github.com/yang991178/fluent-reader)** - 开源跨平台阅读器（Windows/Mac/Linux）
- **[Feedly](https://feedly.com/)** - 在线 RSS 服务
- **[Inoreader](https://www.inoreader.com/)** - 功能强大的在线阅读器
- **[NetNewsWire](https://netnewswire.com/)** - macOS/iOS 原生应用

### 2. 添加订阅

1. 复制上面的订阅地址
2. 在 RSS 阅读器中选择"添加订阅"或"添加 Feed"
3. 粘贴地址并确认
4. 开始接收博客更新！

## ⚠️ 注意

**当前处于开发模式**，RSS 文件尚未生成。

要查看实际的 RSS 内容，请运行：

```bash
npm run build
```

构建完成后，RSS 文件将位于 `docs/.vitepress/dist/feed.rss`。
