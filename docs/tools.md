## 📝 音乐工具

### ABC 五线谱编辑器

一个基于 abc.js 的在线五线谱编辑和预览工具，支持实时渲染和播放。

**功能特性：**
- 🎵 实时 ABC 记谱法转五线谱
- 🎹 MIDI 音频播放
- 📐 缩放和平移控制
- 💾 导出为 SVG/PNG/MIDI
- 📚 内置示例曲谱

[访问 ABC Playground →](https://jarodchen.github.io/abc-playground/)
